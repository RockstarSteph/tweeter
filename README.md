# Tweeter Project

Tweeter is a simple, single-page Twitter clone using HTML, CSS, JS, jQuery and AJAX for the front-end. The back-end uses Node, Express and MongoDB.

## Dependencies

- Express
- Node 5.10.x or above
- body-parser
- chance
- md5
- mongodb
